# Todo List en React

Esta es una aplicación de lista de tareas. Permite agregar, completar y eliminar tareas.

## Características
- Agregar nuevas tareas.
- Marcar tareas como completadas.
- Eliminar tareas de la lista.
- Interfaz centrada y fácil de usar.
